# 善见 Agent Demo Script

1. 打开 `/projects`，先讲一句定位：这是给有资质公益机构用的救助项目 Agent，不是个人筹款平台。
2. 在首页展示公众项目卡片、脱敏项目详情、项目进展、AI问项目和帮助意向入口。
3. 指出右上角三个入口：求助申请入口、机构四辨工作台、捐助意向管理。
4. 进入 `/apply` 求助申请入口，展示大病救助材料如何被 AI 结构化，尤其是“受助人真实需要”和“我不会整理材料，先写一段话”。
5. 进入 `/workbench` 机构四辨工作台，说明本地 LLM Base URL 是 `http://127.0.0.1:3000`。
6. 点击“运行四辨审核”，说明本地 LLM 是四辨工作台的大脑；如果本地服务不可用，系统会使用本地规则兜底。
7. 解释 AI 只给证据、风险、优先级和复核清单，最终由机构工作人员决定。
8. 回到首页，在项目详情页打开 AI问项目，询问“这个项目目前最需要什么？”并说明 AI 只基于脱敏事实回答。
9. 点击我要帮助或进入 `/intentions` 捐助意向管理，说明平台只登记钱/物/服三类帮助意向，不在平台内收款。
10. 展示 AI 如何把帮助意向匹配到治疗费用、政策咨询、交通陪诊等真实需要，并生成机构跟进话术。
